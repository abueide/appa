# Contributing to Appa Documentation

Thank you for contributing to the Appa documentation! This guide will help you maintain consistency and quality across all documentation.

## Documentation Structure

Our documentation follows a user journey approach:

- **00-getting-started/** - New user onboarding and tutorials
- **01-user-guide/** - Daily operations and CLI reference
- **02-architecture/** - Design docs and technical deep dives
- **03-plugins/** - Plugin-specific documentation
- **04-reference/** - Technical reference and troubleshooting
- **contributing/** - Guidelines for contributors
